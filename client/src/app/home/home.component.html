Mean-Angular 4 Todo List with Registration-login Example

This a simple example for MEAN Stack with Angular 4, cantains the listed features as below:

  • User registration
  • User login (username: test, password: test)
  • Create a Todo list for login user

Related teches been used are list below:

  • Angular 4
